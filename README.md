# AI安全買い物チェッカー

> 商品URLや説明文を貼るだけで、怪しい点をAIが整理する1画面アプリ。
> 父（AVに詳しくない世代）でも「これは便利」と分かる、安全・実用・かんたんを目指す。

## 何ができる

- 商品URL／説明文を貼って「安全チェックする」
- **安全そう／注意／やめた方がいい** の3段階で判定
- 気になる点（価格・出品者・支払い・発送・あおり文・URL）を最大4つ表示
- 「買う前に確認すること」を提示
- 履歴を localStorage に保存（端末内のみ）

## 大事な前提

- 判定は**ローカルのルールベース（モック）**。外部AI・サーバーには繋がない。
- あくまで**参考情報**。最終判断は本人が行う（画面下に明記）。

## Tech Stack

React 18 + TypeScript / Vite / Tailwind CSS。データは localStorage。

## Setup

```bash
npm install
npm run dev
npm run build   # docs/ に出力（GitHub Pages 配信）
```

## 次に「本物のAI接続」へ進めるなら（3手順）

1. **サーバー（中継）を1つ用意する**：ブラウザから直接AIのAPIを呼ぶとキーが漏れる。Cloudflare Workers などに小さな中継を置き、APIキーは環境変数で持つ（フロントには絶対に置かない）。
2. **判定を関数に差し替える**：今の `judge()`（ルールベース）を、中継APIへ `fetch` する非同期版に入れ替える。入力（URL/説明文）を渡し、`{verdict, score, reasons, hitokoto}` のJSONを返す形は今と同じにすれば、画面はほぼそのまま動く。
3. **安全弁を付ける**：API失敗時は今のルールベース判定にフォールバック／「参考情報・最終判断は本人」を画面に残す／URLは取得せずテキストだけ送る（個人情報・購入情報を送らない）。
